Widget to allow your visitors contact you through your Nabaztag.

== Description ==

If you have a Nabaztag you must have this widget to allow your visitors, firends and family contact you through your Nabaztag.

It's a simple widget made using jQuery, HTML, PHP and Nabaztag's API. You can select the voice, style of the widget...

You need PHP CURL for this Widget.

== Installation ==

1. Upload `wp-nabaztag` directory to your wordpress `/wp-content/plugins/` directory
1. Activate the plugin through the `Plugins` menu in WordPress
1. Go to widget menu to place it and configure it as you like
